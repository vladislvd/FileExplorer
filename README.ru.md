# FileExplorer
Быстрый и легковесный десктопный файловый менеджер.

## Особенности
* Высокая скорость работы
* Возможность работать с разными дисками
* Мгновенный настраиваемый поиск файлов
* Доступно открытие, удаление и сортировка файлов по разным параметрам 
* Минималистичный интерфейс

## Архитектура
* **Интерфейс** реализован на базе библиотеки `egui`
* **Ядро приложения** написано на чистом Rust (управление файловой системой, индексация)

## Использование
### Для пользователей
1. Перейдите в раздел [Releases](https://github.com/vladislvd/FileExplorer/releases).
2. Скачайте архив `Explorer.zip` из последней версии.
3. Распакуйте и запустите `exe` файл.

### Для разработчиков
#### Требования для сборки:
* **[Rust](https://rust-lang.org/)** - последняя стабильная версия.
* **C++ Build Tools** - идут вместе с Visual Studio или Build Tools для VS. При установке Rust через `rustup` он сам предложит скачать Visual Studio Installer и поставить галочку напротив `Разработка классических приложений на C++ (Desktop development with C++)`.
### Сборка и запуск
1. Скопируйте репозиторий:
```bash
git clone https://github.com/vladislvd/FileExplorer.git
```
2. Перейдите в папку проекта:
```bash
cd FileExplorer
```
3. В папке проекта запустите в режиме разработки:
```bash
cargo run
```
или release режиме:
```bash
cargo run --release
```
3. Скомпилированные файлы сборки будут находиться в директории `target/debug/` или `target/release/` соответственно.
